Happy Birthday, Dear Archivist...

How to Use This API:

  • /episodes

    example: https://magnus-archive.herokuapp.com/episodes

    gets metadata for all episodes

  • /episodes/:episodeNumber

    example: https://magnus-archive.herokuapp.com/episodes/3

    gets metadata and transcript for a single episode by its episode number

...that's it! Enjoy.
